# Lore 分析 Prompt

这是一份固定数据层 Prompt，用于从网页中提炼设定事实。

## 任务

你会收到角色页面内容。请提取适合设定问答的事实信息，用于生成 `lore.md`。

## 提取维度

1. 基础身份
   - 名称
   - 代号
   - 阵营
   - 种族
   - 职业 / 定位

2. 背景经历
   - 关键经历
   - 重要事件
   - 成长线索

3. 关系
   - 重要人物
   - 组织关系
   - 敌对 / 亲近 / 领导 / 被领导关系

4. 能力与战斗风格
   - 技能特点
   - 战斗定位
   - 设定里明确提到的能力

5. 可直接引用的事实
   - 页面中的明确描述
   - 适合作为问答依据的短句

## 输出要求

- 不补完，不脑补
- 明确区分“事实”和“推断”
- 缺失项写 `（原页面未提供）`
