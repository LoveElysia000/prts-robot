# Persona 分析 Prompt

这是一份固定数据层 Prompt，用于从网页中提炼角色本体。

## 任务

你会收到一个角色的标准化资料和页面摘录。请只提取“像不像这个角色”相关的信息，用于生成 `persona.md`。

## 重点提取维度

1. 说话风格
   - 句子长短
   - 正式 / 冷静 / 热烈 / 讽刺
   - 是否常用称呼、口头禅、固定句式
   - 回答时是直接、绕弯、克制、锋利，还是先观察再表达
   - 是否习惯反问、提醒、命令、安抚、留白

2. 性格与价值观
   - 核心性格
   - 优先保护什么
   - 对冲突、牺牲、责任、命令的态度
   - 明显坚持的原则
   - 面对压力时更倾向理性压住情绪，还是情绪先露出来

3. 人际互动
   - 对上级
   - 对同伴
   - 对陌生人
   - 对敌人
   - 面对弱者、后辈、需要帮助的人时的态度
   - 面对亲近对象时是否会出现更柔软、嘴硬、强势或回避的一面

4. 情绪反应
   - 平静时如何表达
   - 轻微不满时如何表达
   - 生气时如何表达
   - 失望时如何表达
   - 无奈时如何表达
   - 心软时如何表达
   - 关心别人时如何表达
   - 高压、紧急、危险场景下如何说话
   - 情绪上来时是先变冷、变短、变硬、变直接，还是先沉默观察

5. 情绪升级链
   - 从轻微不满到明显生气，语气通常怎么变化
   - 被踩到底线时，会怎样明确边界
   - 情绪过后，是会主动收束、继续冷着、还是缓慢恢复正常
   - 页面里能否看出这个角色的“吵完之后怎么收”或“火压住后怎么说”

6. 情绪边界
   - 容易被什么触发
   - 哪些行为会明显不符合角色
   - 哪些过度现代化、过度甜腻、过度轻浮的表达会明显出戏

7. 日常对话能力
   - 角色在闲聊时会怎么回应
   - 如何安慰、鼓励、陪伴别人
   - 遇到非设定类问题时，会用什么态度和语气回答
   - 对夸奖、道歉、示弱、求助、玩笑、沉默分别会怎么回应
   - 能否自然表达关心，还是更倾向克制地表达

8. 高频互动线索
   - 页面里是否能看出这个角色在高频场景下的稳定倾向，例如：
     - 别人很累或情绪低落时
     - 别人做错事、冒险或乱来时
     - 别人依赖她、向她求助或道歉时
     - 别人惹她不高兴时
   - 如果页面没有依据，不要补成确定结论

## 输出要求

- 只写有页面依据的内容
- 只提炼“角色本体”，不要引入朋友、恋人、同事等关系身份设定
- 缺失项明确写 `（原页面未提供）`
- 直接产出可用于 `persona.md` 的结构化内容
